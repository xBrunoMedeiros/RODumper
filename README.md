# Ragnarok GRF Dumper
This is a project that provides a DLL to replace the `gepard.dll` in `Ragnarok Online`, allowing for dumping of all loaded GRFs (Graphical Resource Files).

## How to Use
1. Rename the `gepard.dll` file located in the game folder to `gepard.bin`.
2. Place the DLL provided by this project in the game folder, and rename it to `gepard.dll`.
3. When on the login screen, press `F1` to initiate the dumping process.
